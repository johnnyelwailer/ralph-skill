# Frontend Skills

Structured guidance for frontend development: Tailwind CSS, Chakra UI, and implementing Figma designs.

## Install

```bash
tessl install jamesmoss/frontend
```

## Skills

| Skill | Description |
|-------|-------------|
| **[tailwind](./tailwind/)** | Utility-first styling with Tailwind CSS: layout, responsive design, theming, dark mode, and config. |
| **[chakra-ui](./chakra-ui/)** | Building UIs with Chakra UI: layout primitives (Box, Flex, Stack), components, theming, and v2/v3 patterns. |
| **[figma-to-code](./figma-to-code/)** | Implementing Figma designs in code: layout mapping, spacing, typography, colors, tokens, and accessibility. |

## Structure

Each skill lives in its own directory and includes a `SKILL.md` file with instructions and examples.
